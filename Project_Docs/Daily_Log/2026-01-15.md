## 📝 오늘 진행한 작업

### 1. 프론트엔드 프로젝트 초기 설정
- **Vite + React** 기반 프로젝트 생성 (`front/fronted`)
- 필요한 패키지 설치 (`react-router-dom`, `lucide-react`)
- 프로젝트 폴더 구조화
  - `components/common`: 공통 UI (Button, Card)
  - `components/layout`: 레이아웃 (BottomNav)
  - `components/shared`: 도메인 공유 컴포넌트 (MatchCard)
  - `pages/*`: 페이지 단위 컴포넌트
  - `styles/`: 글로벌 스타일

### 2. UI/UX 디자인 시스템 구축
- **다크 테마** 기반의 스타일링 적용
- **Glassmorphism** 효과를 주는 `Card` 컴포넌트 구현
- 직관적인 그라데이션 컬러 팔레트 정의
  - Primary: Purple (`#667eea`) → Blue Violet (`#764ba2`)
  - Secondary: Pink (`#f093fb`) → Red (`#f5576c`)
