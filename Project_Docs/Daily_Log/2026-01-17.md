6. 오라클 클라우드 서버 배포
   - 백엔드(FastAPI) 및 프론트엔드(React/Vite) 코드 SCP 전송 완료.

   - 프론트엔드 실행: npm run dev (3000 포트)

7. 트러블슈팅 및 환경 설정
   - Node.js 버전 업그레이드: Vite 최신 버전 호환성을 위해 v18 -> v20으로 교체.
   - 방화벽(Firewall) 개방:
     A. 서버 내부(OS): iptables 명령어로 3000, 8080 포트 개방.
     B. 클라우드 외부(Oracle): Ingress Rules에 포트 추가.
   - API 주소 분기 처리 (Local vs Prod):
     A. client.js 코드를 수정하여 환경변수(VITE_API_URL)를 우선 사용하도록 변경.
     B. 서버 .env 파일에 실제 공인 IP 주소(http://168.107.52.201:8080) 등록.
     C. 이를 통해 로컬에선 localhost, 서버에선 공인 IP로 자동 연결되도록 구성.
