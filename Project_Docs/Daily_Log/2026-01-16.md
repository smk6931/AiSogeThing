1. 유튜브 API 연동 및 고도화
- YouTube Data API v3 연동: 검색(Search) 및 인기 영상(Popular) 조회 기능 구현.
- UI/UX: 'Youtube Lounge' 페이지 제작, 비디오 플레이어 모달 구현, 카테고리별 탭(ID 1~30) 구성.
- 쇼츠(Shorts) 제어 로직:
  - 영상 길이를 분석하여 쇼츠 여부 판별.
  - 리스트 내 비율에 따라 쇼츠/일반 영상 자동 정렬 알고리즘 적용.
  - 유저 편의를 위한 '쇼츠 숨기기' 토글 버튼 추가.

2. API 비용 최적화 전략 (Cost Saving)
- 할당량(Quota) 관리 시스템: 일일 10,000 포인트 제한을 로컬 파일(json)로 실시간 카운팅 및 시각화.
- RSS 피드 크롤링 도입 (Zero Cost):
  - 특정 유튜버(연애, 코칭 등)의 최신 영상을 API가 아닌 RSS XML 파싱으로 수집.
  - 비용 '0원'으로 타겟 영상 무제한 제공 구조 마련.
- 채널 리스트 외부화: 하드코딩된 채널 정보를 `dating_channels.json`으로 분리하여 관리 효율성 증대.

3. AI 채널 발굴 기능 (v1)
- 사용자가 버튼 클릭 시 백엔드에서 관련 유튜버를 검색하여 리스트에 자동 추가하는 기능 구현.
- '자가 증식형' 플랫폼 발판 마련.
