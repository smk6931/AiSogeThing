=== 유튜브 콘텐츠 및 큐레이션 전략 (Youtube Strategy) ===

1. 핵심 목표
   - **개인화:** 사용자가 직접 관심사를 입력하고 채널을 발굴하여 나만의 피드를 구성.
   - **비용 효율성:** 고비용의 YouTube Data API 사용을 최소화하고, 무료인 RSS 피드를 적극 활용.
   - **사용자 경험:** 즉각적인 반응(Optimistic UI)과 직관적인 구독 관리(Sidebar) 제공.

2. 데이터 수집 및 비용 최적화 전략 (Cost Zero Strategy)

   A. 채널 발굴 (Discovery) - Cost: 100/회
      - YouTube Data API (Search: Channel) 사용.
      - **전략:** API 호출 시 `maxResults=50`으로 최대한 많은 후보군 확보 (비용은 100으로 동일).
      - **검증단계:** 백엔드에서 즉시 각 채널의 RSS 피드를 조회하여 '최신 영상 유무' 확인.
      - **필터링:** 활동이 없는(RSS Empty) 채널은 자동 제외. 활동 중인 알짜 채널 15개만 선별하여 반환.
      - **결과:** 사용자는 "100% 영상이 존재하는" 채널만 보게 됨.

   B. 영상 로딩 (Video Fetching) - Cost: 0
      - YouTube Data API를 전혀 사용하지 않음.
      - 발견된 채널 ID를 이용해 `https://www.youtube.com/feeds/videos.xml?channel_id={ID}` 호출.
      - XML 파싱하여 영상 메타데이터(제목, 썸네일, 날짜) 추출.
      - **장점:** 무제한 호출 가능, 비용 0, 별도 인증 불필요.

   C. 구독 관리 (Subscription) - Database 사용
      - 기존: 발굴 즉시 DB 저장 -> 스팸성 데이터 누적 문제.
      - **개선:** 발굴은 임시(Memory) -> 사용자가 "구독" 버튼 클릭 시에만 DB(`user_subscriptions`) 저장.
      - **효과:** DB 용량 절약 및 사용자 취향 정밀 분석 가능.

3. 시스템 아키텍처 (Architecture)

   [Client (React)] --(1. 키워드 검색)--> [Backend (FastAPI)]
                                              |
                                     (2. API 검색 (50개))
                                              v
                                       [YouTube Data API]
                                              |
       <--(3. 후보 채널 50개 반환)-------------+
       |
       +--(4. RSS 검증 (Loop))--> [YouTube RSS Server]
       |                              | (무료 XML)
       <--(5. 활동 채널 15개 선별)-----+
       |
   [Client] <--(6. 최종 채널 리스트 반환)-- [Backend]

   [Client] --(7. 구독 클릭)--> [Backend] --> [DB (user_logs)] : 구독 정보 저장
   [Client] --(8. 사이드바 클릭)--> [Backend] --> [RSS] : 해당 채널 최신 영상 로드

4. UI/UX 디자인 전략

   A. 발굴 및 구독 흐름
      - 메인 화면: 발굴된 채널은 '칩(Chip)' 형태로 표시.
      - 구독 액션: 채널 칩 옆 '구독' 버튼 또는 영상 카드 하단 '구독하기' 버튼.
      - 피드백: 구독 클릭 시 리스트에서 즉시 사라지고(숨김 처리), 우측 사이드바로 이동 애니메이션.

   B. 사이드바 (My Subscriptions)
      - 우측 하단 플로팅 버튼(❤️)으로 언제든 접근 가능.
      - 내가 구독한 채널 리스트만 깔끔하게 표시.
      - 채널명 클릭 시 -> 해당 채널의 영상들만 메인 화면에 로드.
      - 삭제(X) 버튼으로 손쉽게 구독 취소.

5. 향후 확장 계획
   - **AI 분석:** `user_subscriptions`에 쌓인 채널들의 성향을 Gemini로 분석하여 '사용자 페르소나' 자동 생성.
   - **자동 추천:** 사용자가 검색하지 않아도 페르소나에 맞는 채널을 매일 1개씩 자동 발굴하여 추천.
