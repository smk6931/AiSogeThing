Q. Backend Framework: FastAPI vs Django?

후보 1: FastAPI (추천)
   - 특징: 매우 빠름(Async), 배우기 쉬움, 최신 파이썬 트렌드.
   - 장점: 
     : AI/ML 라이브러리와 비동기 호환성 최상.
     : 가볍게 시작해서 확장하기 좋음 (MSA).
     : Swagger UI 자동 생성.
   - 단점: 기본 기능(Auth, Admin 등) 직접 구현 필요.

후보 2: Django (DRF)
   - 특징: "Batteries Included", 강력한 관리자 페이지.
   - 장점: 회원가입/로그인, 관리자 페이지 기본 제공.
   - 단점: 무거움, 비동기 처리가 FastAPI만큼 매끄럽지 않음.

💡 추천 의견
   "AI 소개팅" 특성상 실시간 채팅, AI 연산 대기가 많으므로 "FastAPI" 추천.
