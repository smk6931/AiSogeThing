1. AI 기반 '썰' 콘텐츠 자동 생성 시스템
- 개요: 커뮤니티 내 인기 게시글(연애 썰, 소개팅 후기 등)을 바탕으로 2차 콘텐츠를 AI가 자동 생성
- 소스 데이터:
  1) 연애 게시판 베스트 글
  2) 소개팅 성공/실패 썰
  3) 일반 유머/썰 게시글
- 생성 결과물:
  1) 쇼츠(Shorts) 대본 및 영상: 텍스트를 분석하여 나레이션 대본 작성 및 배경 영상 합성
  2) 짤(Meme) 이미지: 상황에 맞는 짤방 생성
  3) 요약형 카드뉴스: 긴 글을 3줄 요약 또는 이미지 텍스트로 변환
- 기대 효과:
  1) 커뮤니티 활성화 (내 글이 영상이 되는 재미)
  2) 앱 내 볼거리 풍성 (유튜브/인스타 스타일)
  3) 외부 바이럴 마케팅 소스로 활용 가능
