1. AI 기반 '썰' 콘텐츠 자동 생성 시스템
- 개요: 커뮤니티 내 인기 게시글(연애 썰, 소개팅 후기 등)을 바탕으로 2차 콘텐츠를 AI가 자동 생성
- 소스 데이터:
  1) 연애 게시판 베스트 글
  2) 소개팅 성공/실패 썰
  3) 일반 유머/썰 게시글
- 생성 결과물:
  1) 쇼츠(Shorts) 대본 및 영상: 텍스트를 분석하여 나레이션 대본 작성 및 배경 영상 합성
  2) 짤(Meme) 이미지: 상황에 맞는 짤방 생성
  3) 요약형 카드뉴스: 긴 글을 3줄 요약 또는 이미지 텍스트로 변환
- 기대 효과:
  1) 커뮤니티 활성화 (내 글이 영상이 되는 재미)
  2) 앱 내 볼거리 풍성 (유튜브/인스타 스타일)
  3) 외부 바이럴 마케팅 소스로 활용 가능

2. 유튜브 연동 확장 (기존 계획)
- 시청 기록 기반 취향 매칭 (RAG)
- 플레이리스트 공유 및 소통 공간 (Youtube Lounge)

3. 유튜브 기반 '취향 공동체' 형성 및 매칭 데이터화 (Interest Graph)
- 아이디어:
  - 단순히 영상을 보는 것을 넘어, 특정 영상(URL)마다 우리 앱만의 '댓글 채널'을 생성.
  - "같은 영상을 보고, 같은 감정을 느낀 사람은 최고의 매칭 상대다."
- 정책 검토 (Policy Check):
  - 유튜브 플레이어 하단/측면에 별도의 댓글창을 운영하는 것은 '임베딩 사이트의 고유 기능'으로 인정됨 (합법).
  - 영상 자체를 가리거나 변형하지 않는다면 문제없음. (레딧, 인벤 등 사례 다수).
- 데이터 활용 방안:
  - User A와 User B가 '하트시그널 레전드편'에 둘 다 긍정적인 댓글을 남김 -> 매칭 점수 가산.
  - 연애 유튜버(김달 등)의 특정 조언 영상에 공감함 -> 가치관의 일치 판단 근거로 활용.
- 기술적 구조:
  - DB Table: Comments (video_id를 Key로 사용)
  - 이 데이터는 향후 RAG(검색 증강 생성) 매칭 시스템의 핵심 '취향 데이터'로 활용됨.
