# 터미널 명령 확인 스킬 (Safe Run)

에이전트가 직접 명령어를 실행하지 않고 유저의 검토를 거치는 프로세스입니다.

## 1. 지침 (Instructions)
1. **명령어 제안**: 실행할 명령어를 코드 블록으로 제시하고 용도를 설명합니다.
2. **복사 안내**: 유저가 직접 터미널에 복사하여 실행하도록 안내합니다.
3. **승인 대기**: 유저가 "실행 완료" 혹은 "확인"이라고 답할 때까지 다음 단계로 넘어가지 않습니다.
4. **가상환경 준수**: Python 명령어는 반드시 `.\venv\Scripts\python` 형식을 포함합니다.

## 2. 예시
- "아래 명령어를 복사하여 실행해 주세요:"
  ```powershell
  .\venv\Scripts\pip install pandas
  ```
- "실행이 완료되면 말씀해 주세요."
